## Magic

### 参数说明

::: tip Magic[n,dim:2]
- n 类型为 `Integer`, 表示幻方的阶
- dim 类型为 `Integer`, 表示幻方的维度
:::

### 可选项

待定, 支持其他种类幻方

### 返回值

`MagicObject`

::: danger Magic
讲道理要不是 matlab 里这么叫的, Magic 这种说一半的做法是不可能作为函数名的.
:::
